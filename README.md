# Notebook

\# Notebook System (OOP Project)



\## Overview

This project implements an Object-Oriented system centered around a \*\*Notebook\*\* object. The system simulates a small stationery store that sells notebooks and pens to customers.



\## Class Schema

\*\*Classes:\*\*

\- `StationeryItem` (Base class)

\- `Notebook` (Child of StationeryItem)

\- `Pen` (Child of StationeryItem)

\- `Customer`

\- `Store`
